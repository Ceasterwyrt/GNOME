# Wyrtbox-GNOME
My personal configuration for GNOME.

Place the 'Windows' folder inside ~/.themes and load it through Tweaks.

Gnome Shell Extensions List:
- ArcMenu
- Blur my Shell
- Dash to Panel
- Date Menu Formatter
  - kk:mm\ndd/MM/y
- Rounded Window Corners
